# 🎂 Happy Birthday Website 🎉

A beautiful, emotional, and interactive **Happy Birthday Website** made with **HTML, CSS, and JavaScript**.  
This project is perfect for wishing your **best friend, girlfriend, boyfriend, sister, brother, or someone special** uniquely and memorably.

Instead of sending a simple birthday message, you can send a personalised website filled with love, animations, memories, wishes, and surprises. 💖
